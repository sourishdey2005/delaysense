# **App Name**: DelaySense: Airline Operations Assistant

## Core Features:

- Flight Delay Prediction: Predict whether a flight will be delayed or on time based on flight details using a trained ML model.
- Input Form: Collect user inputs for flight details such as airline, duration, departure time, weather, congestion, distance, origin and destination airports.
- Prediction Results Display: Display the prediction results (Delayed/On Time) along with a probability score in a modern card UI.
- Airport Delay Contribution Chart: Display a bar/heatmap chart of the top airports causing delays.
- Delay by Hour Chart: Visualize delays by time of day using a line/bar chart.
- Feature Importance Display: Show feature importance from the trained ML model. A tool to display reasoning from the LLM, deciding what constitutes importance and presenting the analysis.
- Synthetic Dataset Generation: Automatically generate a synthetic dataset for training the ML model.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to evoke trust and technological prowess, referencing the sky and aviation.
- Background color: Dark navy (#1A202C), offering a clean, minimal dark theme.
- Accent color: Electric blue (#00FFFF), drawing focus to CTAs and important information, akin to aviation displays.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for both headlines and body text
- Use aviation-themed icons to represent different flight parameters.
- Implement a clean, modern, minimal dark theme inspired by aviation dashboards with responsive design.
- Incorporate Lottie animations for 'Takeoff' and 'Delay' to provide visual feedback.
